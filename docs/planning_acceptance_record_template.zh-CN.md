# Planning Acceptance 记录模板

## 模板用途

这份模板用于统一每次 planning acceptance 应记录的内容，包括：

- 运行环境
- 验收步骤
- 证据与结论
- 异常与后续动作

## 何时使用

- 新建一次 acceptance 记录时
- 需要把人工验收结果结构化时

模板原文见 [planning_acceptance_record_template.md](planning_acceptance_record_template.md)。
