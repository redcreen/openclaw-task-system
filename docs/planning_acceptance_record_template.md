# Planning Acceptance Record Template

## Purpose

Use this template to record one real or semi-real planning acceptance run.

For full Chinese detail, see [planning_acceptance_record_template.zh-CN.md](planning_acceptance_record_template.zh-CN.md).

## Record At Least

- environment and revision
- sample user input
- automation pre-check results
- user-visible observations
- ops-side observations
- pass or fail judgment
- follow-up actions
