# Planning Acceptance 提交拆分建议

## 目的

这份文档记录一次 planning acceptance 工作的建议提交拆分，方便回看当时怎样把改动切成更稳的 commit。

## 使用方式

- 当你只需要当前项目怎么继续做，不要先读这份
- 当你要回看某次 acceptance 期间建议的提交边界，再回到这里

完整细节见 [planning_acceptance_commit_plan.md](planning_acceptance_commit_plan.md)。
